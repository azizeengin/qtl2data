## Raw data for Gnan et al (2014)

This directory contains raw data from [Gnan et al.
(2014)](https://doi.org/10.1534/genetics.114.170746). The `Makefile`
one directory up also includes instructions to download these genotype
and phenotype data. Note that the supplemental files at Genetics end
`.xls` but they need to be changed to `.xlsx`.
